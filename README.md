# Expense_Tracker_Python
A Python-based command-line tool to track daily expenses and save them to a local CSV file.
## Features
- **Persistent Storage**: Saves data to `expenses.csv` so you don't lose records.
- **Categorization**: Tracks spending across Food, Travel, Personal, and Savings.
- **Error Handling**: Validates user input to prevent crashes.

## How to Run
1. Ensure you have Python installed.
2. Download `expense_tracker.py`.
3. Run the script via terminal: `python expense_tracker.py`

## CSV Format
The data is stored in a simple two-column format:
| Category | Amount |
| :--- | :--- |
| food | 50 |
| travel | 20 |
